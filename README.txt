HOW TO CUSTOMIZE


1. PHOTOS

Put your photos inside:

assets/images/


Rename them:

photo1.jpg
photo2.jpg
photo3.jpg

...

photo25.jpg



2. MUSIC

Put your song inside:

assets/music/


Rename it:

birthday.mp3



3. TEXT EDITING

Open:

config.js


Change:

- Name
- Birthday wish
- Letter
- Theme
- Signature



4. THEMES AVAILABLE

galaxy
sakura
luxury
ocean
rose
lavender
midnight



5. OPEN WEBSITE

Open:

index.html



6. SETTINGS

Open:

settings.html


Use it to create a new config.js file.
